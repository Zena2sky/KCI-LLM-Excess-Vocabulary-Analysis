# LLM 상용화 이후 한국 인문사회과학 학술 초록의 어휘 변화

이 리포지터리는 아래 논문의 분석을 재현하기 위해 필요한 자료를 포함합니다.

구슬, 김가연, 김병준. (2026). 「LLM 상용화 이후 한국 인문사회과학 학술 초록의 어휘 변화: 2004~2024년 KCI 초록 데이터 기반 분석」. 『언어와 정보』, 30(1), 1-24. https://doi.org/10.29403/LI.30.1.1

## 개요

이 분석은 과잉 어휘(excess vocabulary) 접근을 활용하여 한국 인문·사회과학 학술 초록에서 나타나는 문체적 변화를 분석합니다. 

## 리포지터리 구성

/code  
영어 초록 분석 코드, 한국어 초록 분석 코드를 포함합니다.  

/figures  
분석 과정에서 생성된 시각화 결과를 포함합니다.

/excess_word_lists  
영어 및 한국어 초록의 과잉 어휘 목록 파일을 포함합니다.  

## 데이터 이용 안내

논문에 기술된 절차를 따라 공개된 KCI 초록 데이터를 활용하여 분석을 재현할 수 있습니다.

KCI 초록 원본 데이터는 KCI 정보 이용 서약에 따라 제공받은 자가 본인 이외의 자에게 양도나 대여 불가능합니다. 연구 재현을 원할 경우 연구자가 직접 KCI에서 데이터를 신청 혹은 다운로드 가능합니다.

### 약어: SS = 사회과학(Social Sciences), H = 인문학(Humanities).
