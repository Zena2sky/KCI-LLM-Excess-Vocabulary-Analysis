# 재현 자료

이 리포지터리는 논문에서 보고된 분석을 재현하는 데 필요한 자료를 포함합니다.

## 개요

이 분석은 과잉 어휘(excess vocabulary) 접근을 활용하여 한국 인문·사회과학 학술 초록에서 나타나는 문체적 변화를 분석합니다. 2024년의 단어 빈도를 2021, 2022년 선형 추세에 기반하여 예측한 빈도와 비교하였습니다. 다음 두 가지 지표를 산출합니다:

- ratio (p/q)
- delta (p − q)

여기서:
- p = 2024년 실제 빈도  
- q = 2024년 예상 빈도

## 리포지터리 구성

/code  
영어 초록 분석 코드, 한국어 초록 분석 코드를 포함합니다.  

/figures  
분석 과정에서 생성된 시각화 결과를 포함합니다.

/excess_word_lists  
영어 및 한국어 초록의 과잉 어휘 목록 파일을 포함합니다.  

## 데이터 이용 안내

논문에 기술된 절차를 따라 공개된 KCI 초록 데이터를 활용하여 분석을 재현할 수 있습니다.
KCI 원자료는 본 리포지터리에서 제공되지 않습니다.  

### 약어: SS = 사회과학(Social Sciences), H = 인문학(Humanities).
