# LLM 상용화 이후 KCI 학술 초록의 과잉어휘 분석

본 리포지터리에서는 한국학술지인용색인(KCI) 데이터(2004–2024)를 대상으로 수행한 한국 인문·사회과학 학술 초록 분석 연구의 코드와 부가 자료를 제공합니다.

## ✍️ 관련 연구 

본 리포지터리는 다음 두 연구(학술지 게재 논문, 석사학위논문)와 관련됩니다.

1. 구슬, 김가연, 김병준. (2026). 「LLM 상용화 이후 한국 인문사회과학 학술 초록의 어휘 변화: 2004~2024년 KCI 초록 데이터 기반 분석」. 『언어와 정보』, 30(1), 1-24. https://doi.org/10.29403/LI.30.1.1
2. 구슬. (2026).「LLM 상용화 이후 한국 인문사회과학 학술 초록의 과잉어휘 현상과 학술 글쓰기의 변화: 2004~2024년 KCI 데이터 분석」, 석사학위청구논문, 한국학중앙연구원 한국학대학원. 

## 📁 리포지터리 구성

/code  
영어 초록 분석 코드, 한국어 초록 분석 코드를 포함합니다.  

/figures  
분석 과정에서 생성된 시각화 결과를 포함합니다.

/excess_word_lists  
영어 및 한국어 초록의 과잉 어휘 목록 파일을 포함합니다.  

## 🔍 데이터 이용 안내

논문에 기술된 절차를 따라 공개된 KCI 초록 데이터를 활용하여 분석을 재현할 수 있습니다.

KCI 초록 원본 데이터는 KCI 정보 이용 서약에 따라 제공받은 자가 본인 이외의 자에게 양도나 대여 불가능합니다. 연구 재현을 원할 경우 연구자가 직접 KCI에서 데이터를 신청 혹은 다운로드 가능합니다.
