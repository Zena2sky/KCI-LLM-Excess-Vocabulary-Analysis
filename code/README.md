## Code

본 폴더에는 KCI 인문·사회과학 초록(2004–2024)을 대상으로 수행한

전처리, 언어 식별, 과잉어휘 분석 및 초록 단위 확산 분석에 사용된 코드가 포함되어 있습니다.

#### 1. 한국어 초록 언어 감지

- `Korean_language_detection_SS.ipynb`
- `Korean_language_detection_H.ipynb`

`langdetect`를 이용하여 한국어 초록을 식별하고 최종 분석 코퍼스를 구축하였습니다.

※ 사회과학(SS), 인문학(H)은 동일한 절차를 사용하며, 입력 데이터만 다릅니다.


#### 2. 단어 수준 분석 

- `English_analysis.ipynb`

  영어 초록 과잉어휘 분석 

※ 사회과학과 인문학은 동일한 분석 절차를 사용하며, 입력 데이터 변경을 통해 재현 가능합니다.

- `Korean_analysis.ipynb`

  한국어 초록 과잉어휘 분석

※ 영어 분석과 동일하나 형태소 분석(Kiwi CoNg) 및 복합 패턴 복원 단계를 추가하였습니다.

#### 3. 초록 단위 확산 분석

- `KorEng_abstract_unit_analysis.ipynb`

  평균·중앙값 및 분포 이동 분석

  ※ 해당 코드는 영어/한국어 및 사회과학/인문학 데이터에 공통으로 적용됩니다.

### 재현성 관련 사항

- 한국어 초록은 langdetect를 이용하여 한국어(`ko`)로 판별된 초록만 분석에 포함하였습니다.
- 모든 시각화 결과는 `/figures` 폴더에 정리되어 있습니다.
- 최종 과잉 어휘 목록은 `/excess_word_lists` 폴더에 정리되어 있습니다.
